# WeatherNow 🌤️

> App climatico com geolocalizacao automatica, previsao de 7 dias e mapas de calor.

## Stack
React • OpenWeather API • Geolocation API • Chart.js

## Funcionalidades
- Deteccao automatica de localizacao
- Previsao detalhada para 7 dias
- Graficos de temperatura e umidade
- Alertas de condicoes extremas
- Animacoes meteorologicas

## Metrica
Dados em tempo real de mais de 200.000 cidades ao redor do mundo.
